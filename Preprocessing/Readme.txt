Les dejo los 4 archivos .py para realizar el preprocesamiento, numerados por paso, y explicados en los comentarios.

Cada funcion tiene parametros de Origin y Destination para indicar donde esta la data de input y a donde se guarda el output.

Para que funcione con los valores de Origin y Destination por defecto los 4 archivos tienen que estar en una carpeta, que a su vez contenga todas las subcarpetas correspondientes a los foros (una subcarpeta por foro, cada subcarpeta tiene que contener el archivo Posts.xml correspondiente a ese foro)

Dentro de cada subcarpeta se van a guardar los archivos propios de cada foro, y fuera de ellas, los archivos globales.

Asegurense de no crear ninguna subcarpeta adicional que no sea de un foro.
